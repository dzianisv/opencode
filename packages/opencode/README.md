# js

To install dependencies:

```bash
bun install
```

To run the local development version:

```bash
bun run dev
```

## Local Installation

To compile and install the `opencode` CLI to your local directory (`~/.local/bin`), run:

```bash
bun run install:local
```

Make sure `~/.local/bin` is in your `PATH`. Alternatively, you can use `bun link` to symlink the package globally:

```bash
bun link
```

This project was created using `bun init` in bun v1.2.12. [Bun](https://bun.sh) is a fast all-in-one JavaScript runtime.
